# 发明名称

一种多智能体目标追逃方法、系统、设备及存储介质

# 说 明 书 摘 要

本发明公开了一种多智能体目标追逃方法、系统、设备及存储介质，属于多智能体协同控制与强化学习技术领域。所述方法将多追捕智能体与单逃逸目标建模为连续动作空间下的马尔可夫博弈过程，构建包含自身状态、邻居关系、目标相对状态及共享记忆的局部观测；建立由基础追捕奖励、连续压制奖励、捕获奖励、碰撞惩罚、速度正则项及围捕几何奖励构成的复合奖励函数；在围捕几何奖励中，通过目标周向阻塞建模求取最大潜在逃脱夹角，并分别计算追捕侧与逃逸侧方向得分；采用角色共享训练机制对追捕策略进行迭代优化，并在评估阶段采用固定任务与恢复式初始化机制进行可复现实验评估。该方案能够在局部观测和可变追捕规模条件下提升围捕协同质量，降低同伴干扰，提高捕获效率与评估稳定性。


# 权 利 要 求 书

1. 一种多智能体目标追逃方法，其特征在于，包括：
S1、构建追逃环境，所述追逃环境包括多个追捕智能体和一个逃逸目标，定义各智能体在连续动作空间中的动作输入，并基于预设动力学模型进行状态更新；
S2、构建各智能体的局部观测，所述局部观测至少包括自身状态特征、同阵营邻居特征、目标相关特征以及目标共享记忆特征；
S3、构建复合奖励函数，所述复合奖励函数至少包括基础追捕奖励项、连续压制奖励项、捕获奖励项、碰撞惩罚项和速度正则项；
S4、基于逃逸目标周向阻塞关系计算最大潜在逃脱夹角，并依据所述最大潜在逃脱夹角构建围捕几何奖励项；
S5、基于所述局部观测和所述复合奖励函数，采用多智能体强化学习算法对追捕策略进行迭代训练，得到训练后的追捕策略模型；
S6、基于固定任务集对所述追捕策略模型进行评估，并输出捕获率、捕获步数和累计奖励指标。

2. 根据权利要求1所述的方法，其特征在于，S1中所述动作空间为二维连续动作向量，控制模式包括速度控制模式和加速度控制模式；在速度控制模式下设置单步最大转角约束，且仅在速度高于预设阈值时启用所述单步最大转角约束。

3. 根据权利要求1所述的方法，其特征在于，S2中所述同阵营邻居特征采用固定数量邻居槽位进行编码，每一槽位至少包含相对位置、相对速度、归一化距离及有效位标识；邻居数量不足时采用零向量填充。

4. 根据权利要求1所述的方法，其特征在于，S2中所述目标相关特征在追捕智能体侧由团队可见性机制控制；在目标不可见时，所述目标共享记忆特征提供历史目标位置、历史目标速度及记忆年龄。

5. 根据权利要求1所述的方法，其特征在于，S3中所述基础追捕奖励项按照追捕智能体与逃逸目标之间的距离分段计算：当距离小于或等于捕获半径时给予正向接近奖励；当距离大于捕获半径时给予负向远离惩罚；所述连续压制奖励项按照追捕智能体处于捕获半径内的连续步数计算并设置上限。

6. 根据权利要求1所述的方法，其特征在于，S4包括：
S41、筛选位于逃脱分析半径内且处于有效状态的追捕智能体；
S42、将每个有效追捕智能体映射为逃逸目标视角下的角度阻塞区间；
S43、对目标周向角度进行离散并生成阻塞掩码；
S44、基于所述阻塞掩码求取最大连续未阻塞扇区，得到所述最大潜在逃脱夹角及其中心方向；
S45、根据逃逸目标速度方向与所述中心方向之间的方向关系，计算追捕侧方向得分和逃逸侧方向得分，并映射为围捕几何奖励项。

7. 根据权利要求6所述的方法，其特征在于，S45中当逃逸目标速度低于预设速度阈值时不计算所述追捕侧方向得分和所述逃逸侧方向得分。

8. 根据权利要求1所述的方法，其特征在于，S5采用角色共享训练机制，多个追捕智能体共享同一策略网络参数；当逃逸目标策略源被配置为可学习模式时，逃逸目标采用独立角色策略参与联合训练。

9. 根据权利要求1所述的方法，其特征在于，S6中评估阶段采用恢复式初始化机制，包括：依据固定任务参数重置随机种子并恢复环境初始状态，从而实现同一任务在多次评估中的可复现比较。

10. 根据权利要求1所述的方法，其特征在于，还包括区域编队初始化步骤：将多个追捕智能体按阵列槽位在随机区域中心进行投放，并约束逃逸目标初始位置与所述区域中心之间满足最小距离阈值。

11. 一种多智能体目标追逃系统，其特征在于，包括：
环境建模模块，用于构建追逃环境并执行状态转移；
观测构建模块，用于构建局部观测与共享记忆；
奖励计算模块，用于计算复合奖励函数及围捕几何奖励项；
策略训练模块，用于训练角色共享追捕策略模型；
评估模块，用于基于固定任务集执行可复现实验评估；
其中，所述系统被配置为执行权利要求1-10任一项所述的方法。

12. 一种电子设备，包括处理器及存储器，所述存储器中存储有计算机程序，所述计算机程序在所述处理器上运行时，使所述电子设备执行权利要求1-10任一项所述的方法。

13. 一种计算机可读存储介质，其上存储有计算机程序，所述计算机程序被处理器执行时实现权利要求1-10任一项所述的方法。


# 说  明  书

## 一种多智能体目标追逃方法、系统、设备及存储介质

### 技术领域

本发明涉及多智能体协同控制技术领域，具体涉及一种多智能体目标追逃方法、系统、设备及存储介质。

### 背景技术

在多无人系统协同任务中，目标追逃属于典型对抗决策问题。现有技术中，追捕侧常采用基于距离的简单奖励设计或固定规则方法，存在以下缺陷：
1. 多追捕体在靠近目标时易产生同向拥挤，导致路径冲突与碰撞风险上升；
2. 在局部观测条件下，难以形成稳定包围结构，目标仍可通过围捕缺口逃逸；
3. 不同实验轮次间初始化条件不一致，导致评估结果稳定性与可比性不足；
4. 当追捕智能体数量变化时，策略复用能力受限。

因此，亟需一种可在局部观测条件下提升围捕协同质量、支持可复现实验评估并兼容规模变化的多智能体目标追逃方案。

### 发明内容

#### 发明目的

本发明的目的在于提供一种多智能体目标追逃方法，通过围捕几何建模与角色共享强化学习相结合，提升围捕协同效率与评估可复现性。

#### 技术方案

为实现上述目的，本发明采用如下技术方案：
1. 将多追捕智能体与单逃逸目标构建为连续动作空间下的多智能体马尔可夫博弈；
2. 构建由自身、邻居、目标与共享记忆组成的局部观测；
3. 构建包含基础追捕、连续压制、捕获、碰撞、速度正则与围捕几何项的复合奖励；
4. 通过目标周向阻塞建模计算最大潜在逃脱夹角，并以该夹角构建追捕侧/逃逸侧方向奖励；
5. 采用角色共享训练机制训练追捕策略，并在固定任务集上执行恢复式评估。

#### 有益效果

相较于现有技术，本发明至少具有如下有益效果：
1. 可显式表征并压缩目标逃逸缺口，提高围捕闭合度；
2. 可在局部观测条件下保持追捕协同稳定性，降低同伴干扰；
3. 支持追捕规模变化下的策略复用；
4. 通过固定任务与恢复式初始化提高评估可复现性与可比性；
5. 便于工程化输出最优模型与分桶评估结果。

### 具体实施方式

以下结合具体实现对本发明作进一步说明。本实施方式用于解释本发明，而非对本发明保护范围的限定。

#### 实施方式一：追逃环境与状态转移

步骤A1：构建追逃环境，设置多个 Hunter 与一个 Target，定义世界边界、时间步长、最大步数及终止条件。  
步骤A2：接收各智能体连续动作输入，并在速度控制模式或加速度控制模式下进行动力学更新。  
步骤A3：执行碰撞检测、边界约束与存活状态更新。  
步骤A4：更新目标可见性、共享目标记忆和捕获计数器。  
步骤A5：判定是否满足终止条件，包括目标被捕获、目标边界碰撞、有效追捕体全部失活或达到最大步数。

#### 实施方式二：局部观测构建

步骤B1：构建自身观测，包含归一化位置与归一化速度。  
步骤B2：构建邻居观测，按距离选取固定数量同阵营邻居并填充槽位特征。  
步骤B3：构建目标观测，包含相对位置、相对速度、归一化距离与可见位。  
步骤B4：构建共享记忆观测，在目标不可见时输出历史目标状态及记忆年龄。  
步骤B5：可选构建协同摘要观测，包含“是否位于最近Top-K追捕体”与“逃脱分析半径内追捕体数量”。

#### 实施方式三：复合奖励计算

设总奖励为：

R = R_hunter_base + R_target_base + R_streak + R_capture + R_collision + R_speed + R_escape_gap。

其中：
1. `R_hunter_base` 与 `R_target_base` 基于追捕体与目标距离分段计算；
2. `R_streak` 基于捕获半径内连续步数计算；
3. `R_capture` 用于捕获事件激励；
4. `R_collision` 包括安全区风险惩罚与硬碰撞惩罚；
5. `R_speed` 为速度归一化惩罚；
6. `R_escape_gap` 为围捕几何奖励。

#### 实施方式四：围捕几何奖励

步骤C1：筛选逃脱分析半径内有效追捕体，设其编号集合为 H。  
步骤C2：对任一追捕体 i∈H，计算其相对目标方位角及角阻塞半宽，得到阻塞区间。  
步骤C3：对目标周向进行离散化，叠加全部阻塞区间生成阻塞掩码。  
步骤C4：搜索最大连续未阻塞扇区，得到最大潜在逃脱夹角及中心方向。  
步骤C5：依据目标速度方向与中心方向的夹角关系，分别计算追捕方向得分与逃逸方向得分。  
步骤C6：按预设系数将方向得分映射为 `R_escape_gap_hunter` 与 `R_escape_gap_target`，并并入总奖励。

#### 实施方式五：训练与评估

步骤D1：采用角色共享机制训练追捕策略，多个追捕体共享同一策略参数；在可学习目标配置下，目标策略独立训练。  
步骤D2：训练阶段采用初始模式重置，支持任务随机化。  
步骤D3：评估阶段采用固定任务集与恢复式重置，以保证同任务多次评估结果一致。  
步骤D4：输出整体评估指标与按追捕数量分桶指标，并保存奖励最优、捕获率最优及捕获步数最优模型快照。

#### 实施方式六：可选配置示例

在一实施例中，可配置如下参数：
1. 追捕智能体最大数量为5，目标数量为1；
2. 环境最大步数为400，评估步数为600；
3. 目标策略源为学习策略；
4. 开启围捕几何奖励，逃脱分析半径为35.0；
5. 训练算法采用循环式多智能体近端策略优化算法。

所述参数仅为示例，不构成对本发明保护范围的限定。

### 结论

本发明通过围捕几何建模、角色共享训练及可复现实验评估机制的联合设计，能够有效提升多智能体目标追逃任务中的协同围捕能力与工程可用性。
